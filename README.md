# Installation

## Prerequisities
- [ ] Python installed
- [ ] Git Installed

## Commands
Run on your terminal

  ```bash
  git clone [https://github.com/F-Phoe-nix/Employee.git](https://github.com/F-Phoe-nix/Employee.git)
  ```

  ```bash
  cd employee
  ```
Open the `employee.ipynb` in your favourite editor

## Note
Ensure that the dataset and the notebook are in the same directory
